HTML是什么

HTML 是一种最常见的，用来编写网页的描述工具。之所以被称之为“描述工具”是因为它本身只是一个用来描述和编写网页的文本工具，本身并没有包含一个编程语言所需的 变量 判断脚本 等 所以在某种意义上来说，不能被称之为“编程语言”，作者纯粹是更具个人喜好，而将其称之为 语言

HTML的四个字母分别都代表什么

HTML 分别代表 Hyper Text Markup Language

Hyper - 超级;超;超越;更
Text - 文本
Markup - 标记;记录
Language - 语言

连起来的意思就是 超文本标记语言

为什么要用 HTML 而不是其他语言

HTML 是在网上最常见的网页编写方式，它支持Javascript CSS 还拥有多个 样式库以及响应框架。
