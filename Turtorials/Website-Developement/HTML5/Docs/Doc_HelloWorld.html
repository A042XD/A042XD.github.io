第一个程序 Hello, world

本章中，你将学习到HTML的基本框架，你也会学会如何写出一个Hello, world网页程序

开发环境

HTML的开发环境不限于操作系统，所需要的工具有
- 浏览器(以下任意一种) Google Chrome, Microsoft Edge, Firefox Mozilla
- 编程工具(以下任意一种) VSCode, Sublime Text 3, Visual Studio, Jetbrains Webstorm
- 网络(以下任意一种) 无线网，有线网
本教程所有文章的开发环境都将使用 Google Chrome 浏览器，Sublime Text 3 编辑器，和无线网络

创建文件

打开Sublime Text 3
按下 Ctrl + N 创建新文件
按下 Ctrl + S 保存该文件

基本框架

<!DOCTYPE html> 是每个HTML文档都需要的语句，它被写在文档的最开头，告诉浏览器这是一个HTML文档，所以将 <!DOCTYPE html> 写到文档的开头，注意 感叹号必须是英文的感叹号
<html></html>这个标签通常被写在 <!DOCTYPE html> 标签的下面，这是一个描述网页的标签，它负责告诉浏览器网页叫什么，图标在哪里，内容是什么……将<html></html>添加到<!DOCTYPE html>的下面
